# LD27

**Theme**: 10 Seconds

## Compiling

Requires Haxe 3 and openfl.

```bash
haxelib install HaxePunk
haxelib install HaxePunkTmx
```

Then simply:

```bash
openfl test flash
```

Or remplace flash with neko to have a desktop build.
